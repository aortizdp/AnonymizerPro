## üöÄ How Can I Contribute?

### 1. Reporting Bugs
If you find a bug, please create a new **Issue** on GitHub. To help us fix it faster, include:
* A clear and descriptive title.
* Steps to reproduce the problem.
* What you expected to happen vs. what actually happened.
* Screenshots if applicable.

### 2. Suggesting Enhancements
Have an idea to make the NLP engine more accurate? Or perhaps a new file format support?
* Open an **Issue** with the tag `enhancement`.
* Explain why this feature would be useful to most users.

### 3. Improving Translations
We aim to support as many languages as possible. If you find an error in our current translations or want to add a new one:
* Modify the `frontend/translations.py` file.
* Submit a Pull Request with the new language dictionary.
